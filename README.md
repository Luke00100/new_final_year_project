# AI-Powered Customer Support Knowledge Assistant

Next.js frontend for the AI Knowledge Assistant — an intelligent customer support system using LLMs and RAG.

## Tech Stack

- **Frontend:** Next.js 14, React 18, TypeScript, Tailwind CSS
- **Backend:** Python (FastAPI) — coming soon

## Getting Started

```bash
npm install
npm run dev

cd backend
.\venv\Scripts\activate
uvicorn main:app --reload
```

Open [http://localhost:3000](http://localhost:3000)

## Pages

- `/login` — Authentication page
- `/dashboard` — Business intelligence analytics
- `/chat` — Knowledge base chatbot
- `/documents` — Document upload and management
